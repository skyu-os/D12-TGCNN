```python
from dataclasses import dataclass
from typing import Dict, List
import numpy as np

@dataclass
class TrafficPrediction:
    timestamp: int
    prediction_horizon: int  # 分钟
    time_interval: int       # 分钟间隔
    sensor_predictions: Dict[int, np.ndarray]
    
    def get_speed_at_time(self, sensor_id: int, time_offset: int) -> float:
        if sensor_id not in self.sensor_predictions:
            return 0.0
        time_index = time_offset // self.time_interval
        if time_index >= len(self.sensor_predictions[sensor_id]):
            return self.sensor_predictions[sensor_id][-1]
        return float(self.sensor_predictions[sensor_id][time_index])
```

#### 2.1 动态边权管理器
创建 `backend/routing/dynamic_edge_weight.py`:
```python
class DynamicEdgeWeightManager:
    def __init__(self, road_graph, traffic_predictor, sensor_mapper):
        self.road_graph = road_graph
        self.predictor = traffic_predictor
        self.mapper = sensor_mapper
        
    def calculate_edge_weight(self, u, v, key, departure_time, vehicle_type='gasoline'):
        # 1. 获取边的基础信息
        length = self.road_graph.get_edge_length(u, v, key)
        
        # 2. 获取预测速度
        sensors = self.mapper.get_sensors_for_edge(u, v, key)
        predicted_speed = self._get_predicted_speed(sensors, departure_time)
        
        # 3. 计算行驶时间
        if predicted_speed > 0:
            return length / (predicted_speed / 3.6)
        else:
            # 使用默认速度
            return length / (40.0 / 3.6)  # 40km/h默认速度
```

### 阶段3: 时变A*算法 (3-4天)
优先级: ⭐⭐⭐⭐⭐ (最高)

#### 3.1 时变A*实现
创建 `backend/routing/time_dependent_astar.py`:
```python
import heapq
from typing import Optional, Dict

class TimeDependentAStar:
    def __init__(self, road_graph, weight_manager, constraints):
        self.G = road_graph.G
        self.weight_manager = weight_manager
        self.constraints = constraints
        
    def find_path(self, start_node, end_node, departure_time, vehicle_type='gasoline'):
        # 优先队列: (f_score, arrival_time, node, prev_node)
        open_set = [(0, departure_time, start_node, None)]
        came_from = {}
        arrival_times = {start_node: departure_time}
        
        while open_set:
            f_score, current_time, current, prev_node = heapq.heappop(open_set)
            
            if current == end_node:
                return self._reconstruct_path(came_from, current, current_time)
            
            for neighbor in self.G.neighbors(current):
                for key in self.G[current][neighbor]:
                    # 计算动态边权（关键：使用当前时间）
                    edge_time = self.weight_manager.calculate_edge_weight(
                        current, neighbor, key, current_time, vehicle_type
                    )
                    
                    # 计算路口约束
                    constraint_time = self._calculate_constraint_time(
                        current, neighbor, prev_node, current_time
                    )
                    
                    # 到达时间
                    arrival_time = current_time + edge_time + constraint_time
                    
                    # 更新最优到达时间
                    if arrival_time < arrival_times.get(neighbor, float('inf')):
                        came_from[neighbor] = (current, key, current_time, arrival_time)
                        arrival_times[neighbor] = arrival_time
                        
                        # 启发式函数
                        h_score = self._heuristic(neighbor, end_node)
                        f_score = arrival_time + h_score
                        
                        heapq.heappush(open_set, (f_score, arrival_time, neighbor, current))
        
        return None
```

## 常见问题和解决方案

### Q1: TGCN模型加载失败
**解决方案**: 
- 检查模型文件路径
- 验证模型文件完整性
- 确认依赖库版本

### Q2: 动态边权计算过慢
**解决方案**:
- 使用预测缓存
- 预计算常用路径
- 优化数据库查询

### Q3: 时变A*算法结果不稳定
**解决方案**:
- 检查启发式函数
- 验证时间窗口逻辑
- 确认预测数据质量

### Q4: 内存使用过高
**解决方案**:
- 限制缓存大小
- 使用LRU缓存策略
- 定期清理过期数据
