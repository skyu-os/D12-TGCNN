## 📍 使用指南

### 基本操作

1. **选择起点**
   - 点击地图上的任意位置
   - 出现绿色标记🟢

2. **选择终点**
   - 再次点击地图上的另一个位置
   - 出现红色标记🔴

3. **选择优化目标**
   - 最短时间(默认)
   - 最短距离

4. **计算路径**
   - 点击"计算路径"按钮
   - 等待2-5秒
   - 查看规划结果

5. **重置**
   - 点击"重置"按钮清除所有标记

### 高级功能

- **缩放地图**: 鼠标滚轮
- **移动地图**: 鼠标拖拽
- **查看坐标**: 点击标记查看详细信息
- **路径统计**: 自动显示距离和时间

## 🔧 故障排除

### 问题1: 服务无法启动

**解决方案:**
```bash
# 检查端口占用
netstat -ano | findstr :5000

# 如果被占用,终止进程或修改端口
# 在 backend/app.py 中修改: port=5000
```

### 问题2: 依赖包问题

**解决方案:**
```bash
# 重新安装依赖
pip install -r requirements.txt
```

### 问题3: OSM数据下载失败

**解决方案:**
```bash
# 手动下载OSM数据
python backend/graph/osm_parser.py
```

### 问题4: 前端无法连接后端

**解决方案:**
1. 检查后端是否正常运行
2. 打开浏览器控制台(F12)查看错误
3. 确认CORS已启用(已在代码中配置)
